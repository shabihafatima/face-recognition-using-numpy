# Face Recognition System using MLP

A simple Face Recognition System implemented using a Multi-Layer Perceptron (MLP) built from scratch with NumPy.

## Features
- Face recognition using grayscale images
- MLP implemented from scratch
- Image preprocessing with PIL
- Forward propagation and backpropagation
- Face prediction on test images

## Technologies
- Python
- NumPy
- Pillow (PIL)

## Note
This project does not use OpenCV, Scikit-learn, TensorFlow, or PyTorch.
